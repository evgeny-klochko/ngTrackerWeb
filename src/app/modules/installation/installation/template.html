Installation

Welcome. Let's fix some errors.

Let's install the TrackerWeb error tracker in your application. It's super easy to get started. Don't worry, there's lots of options for you to customize this later.

Install the TrackerWeb Library

The TrackerWeb library lives in your web application. Paste this snippet before your other scripts in the of your page.

<!-- BEGIN TrackerWeb --> <script type="text/javascript">window.TrackerWeb = { token: 'f92ca77cef5b4f16b81ded8528191c40' };</script> <script type="text/javascript" src="trackerWeb-agent.js"></script> <!-- END TrackerWeb -->
